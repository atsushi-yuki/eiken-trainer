# EIKEN TRAINER

英検（5級・4級・3級）の **文法・単語の習熟度** を測る学習アプリです。文法は「be動詞」「現在完了」「関係代名詞」のような **文法ターゲット単位** で習熟度を判定するため、どの項目が穴かが一目で分かります。間違いはブラウザ内に記録され、苦手克服のための **学習カリキュラムを自動生成** します。

> リスニング・長文は対象外。文法と単語にフォーカスしています。

## 主な機能

- **文法ターゲット方式** — 各級の文法を項目ごと（各15問のストックから毎回5問をランダム出題）に分割し、ターゲット単位で習得/あと一歩/要強化を判定
- **単語ユニット** — 基礎・応用の各ストック（30問）から毎回12問をランダム出題
- **網羅マップ** — 級ごとに全文法ターゲットの達成状況を色で可視化
- **間違いの記録と復習モード** — 直近で間違えた問題だけを抽出して再挑戦
- **カリキュラム自動生成** — 正答率の低い項目から優先度付きで学習ステップを提示
- **15分でチェック** — 1ターゲット数分、1ユニット約15分

学習記録はブラウザの localStorage に保存されます（サーバー不要・端末内のみ）。

## GitHub Pages へ公開する

このリポジトリには2通りの公開方法が用意されています。

### 方法A：GitHub Actions で自動公開（推奨）

`main` ブランチへ push するたびに自動でビルド・公開されます。

1. このコードを自分の GitHub リポジトリ（名前は `eiken-trainer` を想定）に push
2. GitHub の **Settings → Pages → Build and deployment** で **Source** を **「GitHub Actions」** に変更
3. 以降、`main` への push で `.github/workflows/deploy.yml` が走り、自動公開される

公開URL: `https://<ユーザー名>.github.io/eiken-trainer/`

### 方法B：gh-pages コマンドで手動公開

```bash
npm run deploy
```

`dist/` が `gh-pages` ブランチに公開されます。この場合は Settings → Pages の Source を **「Deploy from a branch」→ `gh-pages`** に設定してください。

### 重要：リポジトリ名を変える場合

`eiken-trainer` 以外の名前にするときは、次の2か所を合わせて変更してください。

- `vite.config.js` の `base: "/eiken-trainer/"`
- `package.json` の `homepage`

これが一致していないと、公開後に画面が真っ白になります（アセットのパスがずれるため）。

## 問題データの追加・編集

問題は `src/data/` 配下に級ごと・種別ごとに分かれています。

```
src/
  bank.js          … 全データを統合するエントリ（級の追加はここ）
  data/
    g5.js g4.js g3.js   … 各級の文法ターゲット
    v5.js v4.js v3.js   … 各級の単語ユニット（基礎/応用）
  App.jsx          … アプリ本体
```

### 文法ターゲットの構造（`g*.js`）

```js
export const G5 = [
  {
    key: "be-aff",                 // 一意なキー
    name: "be動詞（肯定）",          // 表示名
    struct: "主語 + be動詞 + 補語",  // 構造の説明（UIに表示）
    questions: [
      {
        id: "5_be-aff_1",          // 全体で一意なID
        q: "I ___ a student.",     // 問題文
        choices: ["am", "is", "are", "be"], // 選択肢（4つ）
        ans: 0,                    // 正解の添字（0始まり）
        exp: "I には am。"          // 解説
      },
      // … 1ターゲットあたり15問程度を推奨（毎回そこから5問をランダム出題）
    ],
  },
];
```

### 単語ユニットの構造（`v*.js`）

```js
export const V5 = {
  "v-basic": { name: "単語・基礎", questions: [ /* 問題… */ ] },
  "v-adv":   { name: "単語・応用", questions: [ /* 問題… */ ] },
};
```

問題IDは全体で重複しないようにしてください（習熟度の記録キーになります）。
